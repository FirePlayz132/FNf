### Input
- `Left Arrow`: **Left Movement - 1**
- `Right Arrow`: **Right Movement - 1**
- `Up Arrow`: **Up Movement - 1**
- `Down Arrow`: **Down Movement - 1**

- `W Key`: **Left Movement - 2**
- `A Key`: **Right Movement - 2**
- `S Key`: **Down Movement - 2**
- `D Key`: **Right Movement - 2**

- `I Key`: **Up Miss**
- `J Key`: **Left Miss**
- `K Key`: **Down Miss**
- `L Key`: **Right Miss**
